## âš™ï¸ Technical Details

This project demonstrates advanced Unity development skills:

* **Scriptable Objects:** Used heavily for Item Data (Weapons, Potions, Quest Items) to make the game easily expandable.
* **Singleton Pattern:** Implemented for GameManager and UIManager to ensure efficient state management.
* **Event System:** Decoupled architecture using C# Actions/Delegates for interactions between the Player, Inventory, and UI.
* **Animation State Machine:** Complex blend trees for smooth transitions between Idle, Run, and Attack states.
